# Customer Dashboard

The dashboard is your main landing page after sign-in.

## Main Uses

- view current loan balance or qualification
- start a loan request
- start repayment flows
- check account and profile details
- open support when you need help

## Primary Location

- [`/customer/dashboard`](/customer/dashboard)

## Suggested Routine

1. Check your current loan and repayment status.
2. Continue with the next required action (loan or repayment).
3. Open support if something needs assistance.
