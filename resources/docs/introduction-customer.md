# Customer Introduction

Welcome to your customer help center.

This guide focuses only on customer actions and screens.

## Fast Navigation

- Use **Dashboard** from the customer top bar to return home quickly.
- Use **Help** from the customer top bar to open this manual anytime.
- Use the floating `?` button to return to Help from customer pages.

1. Review your dashboard and loan status.
2. Start loan or repayment flows.
3. Update your profile and account details.
4. Follow reset instructions if you forgot your PIN.
