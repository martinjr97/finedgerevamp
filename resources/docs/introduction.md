# Introduction

Welcome to the system user manual.

This manual explains where each feature is located, who can access it, and the normal end-to-end workflow for daily operations.

## Access Rules

Some pages in this manual are only visible to users with matching permissions.

- If you cannot see a section, your role does not currently allow that action.
- Ask a system administrator to assign the required permission(s).

## Fast Navigation

- Use the **Help** button in the top bar.
- Use **Start Tour** in the top bar to launch a guided walkthrough of visible menus.
- Use the floating `?` button at the bottom-right to return to Help.

## Core Workstreams

1. Sign in and review the dashboard.
2. Open the required module from the left menu.
3. Create, review, approve, or export records.
4. Return to dashboard and approvals to monitor status changes.
