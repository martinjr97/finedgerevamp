# Settings

Settings are available to admins with appropriate permissions.

## Main Settings Areas

- General settings: [`/admin/settings/general`](/admin/settings/general)
- Customer registration settings: [`/admin/settings/customer-registration`](/admin/settings/customer-registration)
- Repayment reminder settings: [`/admin/settings/repayment-reminders`](/admin/settings/repayment-reminders)
- Credit score settings: [`/admin/settings/credit-score`](/admin/settings/credit-score)
- Security questions: [`/admin/security-questions`](/admin/security-questions)
- Payment channels: [`/admin/channels`](/admin/channels)
- Product configuration: [`/admin/loan-products`](/admin/loan-products) and [`/admin/loan-rate-types`](/admin/loan-rate-types)

## Recommended Change Process

1. Review current setting values.
2. Confirm the operational impact.
3. Apply changes during controlled windows where possible.
4. Notify affected teams if behavior changes.
5. Validate outcome from dashboard and related flows.

## Access Control Reminder

If a setting is not visible, check role permissions first. Visibility and actions are permission-based.
