# Dashboard

The dashboard is your operational starting point.

## Admin Dashboard

Main uses:

- monitor active companies, customers, and loans
- check pending approvals
- review repayment and disbursement trends
- jump directly to detailed modules from summary cards

Primary location:

- [`/admin/dashboard`](/admin/dashboard)

## Customer Dashboard

Main uses:

- view current loan balance or loan qualification
- see account and KYC status
- start loan or repayment flows
- open support when help is needed

Primary location:

- [`/customer/dashboard`](/customer/dashboard)

## Suggested Daily Routine

1. Check dashboard totals and alerts first.
2. Clear pending approvals.
3. Complete customer and loan processing tasks.
4. Run reports and reconciliations.
